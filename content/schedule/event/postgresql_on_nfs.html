When asked by a client to help them setting up a PostgreSQL cluster with NFS-based storage, my first gut reaction was: don't. But as it turns out, this is not such an unholy combination; as long as you keep in mind the effects different settings have on both the NFS client and NFS server and the expections PostgreSQL has regarding file operations, one can get away with it.

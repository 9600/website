--- title: Implementing JSR 292 in IKVM.NET kind: event speakers: - jeroen_frijters slug: implementing_jsr_292_in_ikvm_net day: Sat start: 12:00 end: 12:00 event_id: 680 ---

IKVM.NET is an implementation of a Java VM on top of Microsoft .NET and Mono. In this talk I'll describe how Java 7's JSR 292 invokedynamic and MethodHandles have been implemented in IKVM.NET.

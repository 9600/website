Track: Free Java Devroom

This talk will cover progress with OpenJDK on ARM. It's basically a progress report, but with some technical detail about the changes needed for:

  • The new Hard FP ABI.
  • ARM 7.

I'll also discuss the issues that arise from the Java Memory Model (also known as JSR 233.) The talk will then be in two halves, one part being very specific to Ed Nevill's ARM Assembler interpreter and JIT, and the other part will cover the ARM more generally.

If there's any time left I'll also cover the upcoming ARM8 architecture.

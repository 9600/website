--- title: Getting Started With Ogre3d For Game Development kind: event speakers: - erik_ogenvik slug: ogre3d day: Sun start: 14:30 end: 14:30 event_id: 534 ---

Track: Open Source Game Development Devroom

Learn how to set up and use Ogre3d to provide a powerful and extensible 3d graphical engine. The focus is on explaining the core concepts and objects needed to get rendering up and running, as well as how to integrate the graphical engine with a game.

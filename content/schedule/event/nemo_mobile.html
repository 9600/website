Nemo Mobile is a linux distribution that is based on Mer Core and is using MeeGo Handset UX as its UX at the moment. Probably the easiest way is to think that the Nemo Mobile continues where MeeGo Community Edition (MeeGo CE) left off, with the main difference that it uses Mer as the core instead of MeeGo. In this talk the current status of the project is shown as well as the basics of how community can contribute to the project.

--- title: Analysing LibreOffice with Compiler plugins kind: event speakers: - stephan_bergmann slug: libocompiler day: Sat start: 18:30 end: 18:30 event_id: 755 ---

Track: LibreOffice Devroom

Above all else, LibreOffice is a vast corpus of C++ source code of varying quality. Static analysis of that giant code base guarantees a constant stream of interesting findings. The talk will look at how plugin architectures of modern C++ compilers allow to tap that stream.

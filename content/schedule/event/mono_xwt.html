--- title: XWT kind: event speakers: - lluis_sanchez slug: mono_xwt day: Sat start: 17:45 end: 17:45 event_id: 775 ---

XWT is a new cross platform GUI toolkit for Mono and .NET, which is based on the idea of building an abstract widget library on top of platform-specific native widgets. This toolkit is still in a very early development stage, but there are already backends for GTK+, MonoMac and some initial work on WPF. The talk will explain the goals of the project and some details about design principles and internal architecture.

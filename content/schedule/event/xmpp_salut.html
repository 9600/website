Architecture and use cases of Salut à Toi, a XMPP client with multi frontends (console ncurses-like, command line interface, desktop, web), with special focus on microblogging.

This talk aims to briefly present the Salut à Toi project and its goals. Salut à Toi is a XMPP client which focus on the whole XMPP stuff, not only instant messaging, and where many frontends use a common backend.

The following points will be covered:

  • global overview of the project

  • why a multi-frontends client ?

  • the architecture/technologies used

  • quick presentation of some features

  • XMPP as a replacement of traditional e-mail network

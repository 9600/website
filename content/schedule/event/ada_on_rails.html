Discover how Ada catches up with web application development.

Ada on Rails is a bleeding edge Ada tooled framework environment that enables web applications development. The goal of Ada on Rails is to efficiently use all the strength of the Ada based technologies in web applications, while being: Rapid Application Development compatible; agile friendly; safe and secure; ready for formal method through SPARK Ada; low code, memory and CPU footprint; range from bareboard to large scaled distributed SOA applications.

Bio

London based developer who is the current project co-ordinator of the Open Source MonoGame project. I'm usually a database developer, but with always an interest in game development. I worked on a couple commercial games, namely Siege of Avalon ( now Open Source ) and Hero X. I was also the maintainer of the Object Pascal bindings for SDL, in a time long forgot.

Currently forcused on MonoGame, and trying to write the odd mobile app using MonoTouch and MonoDroid. Current Apps on the App Store Murder Map and Open Mic.
